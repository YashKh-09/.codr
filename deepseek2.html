My Coding Journey

I first tried web development in 2021 with basic HTML, then learned more when my school introduced it in 2022. After a few failed attempts and a short phase with Python and Flask, I got stuck in the tutorial trap. Things changed in August 2025 when I got back into web dev, discovered Web3, and started building instead of just watching tutorials. Learning Tailwind CSS by creating a simple calculator layout gave me real confidence and direction.

Around the same time, I switched to Arch Linux for a clean, distraction-free setup, even though it took hours to install as a beginner. Now, I'm fully focused on building my personal brand and constantly learning ‚Äî from front-end and MERN stack to Web3 and data science. Coding has become something I truly love ‚Äî I can code for hours with music on and still feel calm and fulfilled.
